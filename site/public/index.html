arbos.life

Run an agent anywhere.

Install arbos on any machine, point it at your repo, and get a web UI with tools, terminal, and browser — local or public.

Install & start the web UI

curl -fsSL https://arbos.life | bash -s -- web

Install only

curl -fsSL https://arbos.life | bash

After install

export OPENROUTER_API_KEY=sk-or-...

Then run arbos web for the UI, or cd your-project && arbos in a terminal session.
